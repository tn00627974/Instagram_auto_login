# Instagram_auto_login

此程式碼使用selenium自動登入ig帳號


請確保以下已執行安裝,確保程式碼可運作:


selenium==4.10.0
webdriver_manager
undetected-chromedriver
customtkinter
webdriver_manager


初始先執行app.py
輸入帳號 , 輸入密碼後 ,按下start bot即可
